# Two sum
You are given a list of numbers, and a target number k. Return whether or not there are two numbers in the list that add up to k.

## Example:
    Input [4, 7, 1 , -3, 2] and k = 5,
    Output true 
    Explanation: Since 4 + 1 = 5.
