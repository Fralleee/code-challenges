# Reverse a linked list
Given a singly-linked list, reverse the list. This can be done iteratively or recursively.

## Example:
    Input: 4 -> 3 -> 2 -> 1 -> 0 -> NULL
    Output: 0 -> 1 -> 2 -> 3 -> 4 -> NULL
